===========================Description=============================
This is a small project using whitch you can access any facebook page using python requests which you can access after login in facebook .

===========================Setup=================================== 
1. Open the facebookLogin.py and set your facebook username and password and Save It .
2. Set the proxy to None if not required or change according to your requirement .
3. So to get the login sessions just import facebookLogin module and call the getLoginSession function which provide you the login cookies which helps you access any github page(If everything goes true) or False(if error occcured) .
